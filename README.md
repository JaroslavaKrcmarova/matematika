# Projekt matematika

Tento projekt slouží jako sbírka kódů a příkladů z různých témat matematiky. Každý soubor nebo notebook se váže k jedné konkrétní matematické oblasti a má sloužit jako podklad pro doučování.  

## Cíl projektu
- Poskytnout přehledné a interaktivní materiály pro snadnější pochopení matematických témat.  
- Umožnit studentům procvičování prostřednictvím kódu a příkladů.  
- Sloužit jako doplňkový výukový materiál pro doučování.  

## Obsah projektu
- Každý soubor obsahuje řešené příklady, vysvětlení a kód pro simulace nebo vizualizace.  
- Některé soubory jsou zatím **TODO** – tedy připravené téma, ale kód ještě není hotový.  
- Tematické oblasti zahrnují například:
  - Algebra
  - Geometrie
  - Posloupnosti a řady
  - Funkce a grafy
  - Logaritmy a exponenciální funkce
  - Kvadratické rovnice
  - … a další
